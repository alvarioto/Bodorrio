# **App Name**: Evento Unico

## Core Features:

- Hero Section: Display a welcoming message, names, wedding date, and a short elegant text.
- Countdown Timer: Real-time countdown to the wedding ceremony using JavaScript.
- Ceremony and Celebration Cards: Display detailed information about the ceremony and celebration venues, including location, time and map links.
- Add to Calendar Functionality: Generate and download .ics files for both the ceremony and celebration events using JavaScript.
- RSVP Modal: Interactive modal for guests to RSVP, capture attendance, and submit comments.
- Photo Gallery: Display a curated photo gallery, allowing users to view photos in a light box.
- Event Timeline: Present a timeline of key events throughout the wedding day, from ceremony to after-party.

## Style Guidelines:

- Primary color: Sage green (#A7D189) for main elements, evoking nature and tranquility.
- Light sage green: (#DAE9C8) Used for icon backgrounds and chips.
- Global background color: Very light green (#F4F9F1) with a subtle geometric texture for depth.
- Main text color: Dark grayish-green (#333D36) for readability and elegance.
- Secondary text color: Medium grayish-green (#606B5D) for less critical information.
- Soft border color: Light grayish-green (#C2D1B5) for subtle UI separation.
- Headline font: 'Playfair Display', serif, for titles and important headings, adding elegance and style.
- Body font: 'PT Sans', sans-serif, for body text to ensure readability and clean design.
- Use animated inline SVG icons with consistent line style and themed animations: ceremony rings/hearts, party hat, camera, and timeline elements such as a bus, drinks, and confetti. Follow rules for strokes and fills as requested.
- Cards with rounded-3xl borders, soft shadows, and generous whitespace, all components aligned and centered.
- Subtle hover effects on buttons, including darkening the background and a slight shadow lift. Animations for icons using stroke-dasharray, stroke-dashoffset, transform, and opacity.
